These bias scans are both based on an example data set.

FPR scan: The cluster with highest bias (FPR) deviates 0.08 from the rest of the data set. There are 249 data points in this cluster. This means that, on average, users that:

- are verified, have higher #followers, user engagement and #URLs;

- use less #hashags and have lower tweet length

have more true content classified as false (false positives).

FNR scan: The cluster with highest bias (FNR) deviates 0.13 from the rest of the data set. There are 46 data points in this cluster. This means that, on average, users that:

- use more #hashtags and have higher sentiment score;

- are non-verified, have less #followers, user engagement and tweet length

have more false content classified as true (false negatives).
